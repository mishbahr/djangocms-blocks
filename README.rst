Quickstart
----------

1. Install `djangocms-blocks`::

    pip install djangocms-blocks

2. Add `djangocms_blocks` to `INSTALLED_APPS`::

    INSTALLED_APPS = (
        ...
        'djangocms_blocks',
        ...
    )

3. Then sync and/or migrate::

    python manage.py syncdb
    python manage.py migrate
